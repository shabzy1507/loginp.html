HTML SUMMARY

html stands for hypertext markup language created by tim berners lee

what we need to start

  • A web browser
  • A text editor
  • does not need a server
  • files must have .html extension
  • index.html is the root of a website

HTML FEATURES

Tag name Attribute
(a) tag: href attribute for adding links
html tag: lang attribute for specifying language
img tag: src used to source out .jpg image file/width and height attribute
any tag: style attribute (font-size),(font-family),(color),(background-color),(border),(text-align)
........: style="property:value;"

note:

              inline elements stay in line when writing a code e.g a tag,p tag 
              block elements go to a new line e.g br ,hr,div.
